## このREADMEについて

- 画像は GitHub Actions で定期更新されます。
- 反映まで数分かかる場合があります。
- 表示内容の詳細は `.github/workflows/metrics.yml` で調整できます。

## セキュリティメモ

- トークンは `secrets.GITHUB_TOKEN` を使用し、ワークフロー権限は最小限に設定しています。
- 機密情報をメトリクス設定値に直接書かないでください。
- 公開プロフィールに表示される情報のみを対象に設定してください。
